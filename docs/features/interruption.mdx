## How It Works

Metal's interruption management follows these steps:

1. **Interruption Detection**

   - Monitors system events
   - Identifies potential interruptions
   - Assesses interruption priority

2. **State Management**

   - Captures current work state
   - Preserves important context
   - Prepares for resumption

3. **Recovery Assistance**
   - Provides transition guidance
   - Suggests next steps
   - Maintains work continuity

## Configuration

Customize interruption handling through environment variables:

```bash
# Interruption Settings
METAL_INTERRUPT_DETECT=true
METAL_INTERRUPT_NOTIFY=true
METAL_INTERRUPT_DELAY=5s

# Notification Settings
METAL_NOTIFY_CHANNEL=desktop
METAL_NOTIFY_PRIORITY=high
```

## Usage

Manage interruptions with these commands:

```bash
# Manual interruption
metal interrupt

# View interruption history
metal interrupt --history

# Resume after interruption
metal resume
```

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js 18 or higher
- pnpm 8.15.4 or higher
- Git

## Installation

<AccordionGroup>
  <Accordion icon="github" title="Clone the repository">
    Start by cloning the Metal repository to your local machine: ```bash git clone
    https://github.com/gabrielvfonseca/metal.git cd metal ```
  </Accordion>
  <Accordion icon="package" title="Install dependencies">
    Install all required dependencies using pnpm: ```bash pnpm install ```
  </Accordion>
  <Accordion icon="hammer" title="Build the project">
    Build Metal and all its components: ```bash pnpm build ```
  </Accordion>
</AccordionGroup>

## Running Metal

<AccordionGroup>
  <Accordion icon="play" title="Start development mode">
    To start Metal in development mode: ```bash pnpm dev ``` This will start all services including
    the CLI, daemon, and optional dashboard.
  </Accordion>
  <Accordion icon="gear" title="Configure Metal">
    Metal can be configured through environment variables or a configuration file. See the
    [Configuration](/configuration) section for details.
  </Accordion>
</AccordionGroup>
